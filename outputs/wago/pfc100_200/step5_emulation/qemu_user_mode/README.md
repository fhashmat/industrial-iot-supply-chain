# Step 5: QEMU Emulation Outputs

This folder contains Step 5 emulation evidence for the WAGO PFC100/200 firmware.

The firmware root filesystem was executed using QEMU user-mode emulation on Ubuntu. The outputs include architecture evidence, OS metadata, QEMU execution output, BusyBox runtime tracing, interface/service candidate lists, key binary paths, dependency checks, and service-level traces.

Raw firmware files are not included in the repository.
